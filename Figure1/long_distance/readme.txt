After running `preprocess.ijm` to process the raw image,
1. run `export.m` to get the image in the figure 1a and 1b
2. run `outline.m` to get the outlines in the figure 1a and 1b.
3. run `get_slices.m` to get the figure 1c.
4. run `make_kymograph.m` to get figure 1d (`mask_initiation.m`, `speed_estimation.m`, and `plotKymo.m` are called)
5. run `diffusion_plot.m` to get figure 1e.
(running time ~3m20s)
