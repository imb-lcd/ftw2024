Figure 1a to 1e
1. run `preprocess.ijm` to process the raw image
2. run `outline.m` to get the outlines in the figure 1a and 1b.
3. run `mask_initiation.m` to get the setting for making a kymograph
4. run `speed_estimation.m` to estimate speed based on the kymograph
5. run `make_kymograph.m` to get figure 1d.
6. run `diffusion_plot.m` to get figure 1e.
(running time ~3m20s)
