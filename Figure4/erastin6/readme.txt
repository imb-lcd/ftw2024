After running `preprocess.ijm` to process the raw images,
1. run `export.ijm` to get the images in figure 4d.
2. run `outline.m` to get the outline in figure 4d (`export_for_outline.ijm` to export images for outlines)
3. run `sim_trigger_waves.m` to get the simulated waves shown in figure 4d
4. run `summarize_speed.m` to get the figure 4e.
5. run `measure_wavefront_width_amp.m` to get the figure 4f and 4g.
